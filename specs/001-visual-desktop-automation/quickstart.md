# Quickstart: Visual Desktop Automation Tool

## Installation

1.  **Engine**:
    ```bash
    cd engine
    pip install -r requirements.txt
    python src/main.py --scripts-dir ./my_scripts
    ```
2.  **UI**:
    ```bash
    cd ui
    flutter run -d macos
    ```

## First Steps
1.  Open the UI and navigate to "New Task".
2.  Provide a screenshot of the target element.
3.  Choose an interaction (e.g., Single Click).
4.  Run the task.
5.  Observe the brief visual highlight and the interaction on your desktop.
